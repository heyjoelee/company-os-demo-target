# company-os-demo-target

Throwaway Next.js target repo used to demonstrate the Company OS agent commit → preview loop in the
60-second founder hook demo described in
[JOE-34 beta-outreach-plan §4](https://app.paperclip.ing/JOE/issues/JOE-34#document-beta-outreach-plan).

This repo exists for one purpose: when a Paperclip agent picks up a child issue, it edits files in this
repo, commits, pushes a branch, and Vercel renders a preview URL. The preview is the artifact a viewer sees
in Beat 3 of the storyboard.

If you are not the demo recorder, you should not be here.

Tracked in [JOE-35](https://app.paperclip.ing/JOE/issues/JOE-35).
